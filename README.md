<div><span><font color="#cccccc">

Com hardware e software de fácil utilização. É destinado para todas as pessoa que desejam fazer projetos eletrônicos.</span></p>
<p dir="ltr" style="font-size:medium;line-height:1.2;margin-top:6pt;margin-bottom:0pt"><span style="font-family:Arial;font-weight:400;font-style:normal;font-variant:normal;text-decoration:none;vertical-align:baseline;white-space:pre-wrap;background-color:transparent"><a href="https://www.arduino.cc/">https://www.arduino.cc/</a></span></p>
